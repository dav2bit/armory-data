Military Equipment Data Visualization (/armory-data)Այս պրոյեկտը իրենից ներկայացնում է ռազմական տեխնիկայի և սպառազինության համապարփակ տվյալների շտեմարան, որը նախատեսված է զենքերի տեխնիկական բնութագրերի և մոդելների ուսումնասիրման համար:🚀 Հիմնական առանձնահատկություններըՏվյալների հավաքագրում (Data Scraping & Processing): Օգտագործելով Python սկրիպտներ՝ տարբեր աղբյուրներից արդյունահանվել և վերամշակվել են զենքերի սպեցիֆիկացիաները, 3D մոդելները և պատկերները:Կառուցվածքային տվյալներ: Ամբողջ տեղեկատվությունը կազմակերպված և պահպանված է JSON ֆորմատով, ինչը ապահովում է արագ հասանելիություն և տվյալների թեթև քաշ:Ինտերակտիվ Վիզուալիզացիա: UI-ն կառուցված է JavaScript, HTML և CSS տեխնոլոգիաներով՝ ապահովելով օգտատիրոջ համար հարմարավետ և դինամիկ միջավայր:Որոնման Ալգորիթմ: Պրոյեկտում ներդրված է Search Recommendation (որոնման առաջարկների) ալգորիթմ, որը հեշտացնում է անհրաժեշտ սպառազինության արագ գտնելը:🛠 Տեխնոլոգիական StackBackend/Data: Python (Extraction & Transformation) Frontend: JavaScript, HTML, CSS Storage: JSON 🔗 ՀղումներDemo: https://armory-data.netlify.app/

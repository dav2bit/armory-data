# 🛡️ Military Equipment Data Visualization (/armory-data)

Այս նախագիծը հանդիսանում է ռազմական տեխնիկայի և սպառազինության վերաբերյալ տեղեկատվության համապարփակ շտեմարան, որը նախագծվել է օգտատերերին մանրամասն տեխնիկական պատկերացումներ տրամադրելու համար:

---

## 🚀 Հիմնական Առանձնահատկությունները

* **Data Aggregation (Python):** Օգտագործել եմ Python սկրիպտներ՝ տարբեր աղբյուրներից տվյալների արդյունահանման (extraction) և տրանսֆորմացիայի համար:
* **Structured Storage:** Հավաքագրված ողջ տեղեկատվությունը (սպեցիֆիկացիաներ, 3D մոդելներ, պատկերներ) կազմակերպված և ստրուկտուրավորված է JSON ֆայլի մեջ:
* **Search Recommendation Algorithm:** Ներդրված է որոնման և առաջարկների հատուկ ալգորիթմ, որը հեշտացնում է անհրաժեշտ զենքի կամ տեխնիկայի արագ գտնելը:
* **Interactive Web UI:** Օգտագործվել է JavaScript, HTML և CSS՝ դինամիկ և ինտերակտիվ ինտերֆեյս ստեղծելու համար:

---

## 🛠️ Տեխնիկական Stack

| Տեխնոլոգիա | Կիրառություն |
| :--- | :--- |
| **Python** | Տվյալների հավաքագրում և մշակում |
| **JavaScript** | Ինտերակտիվություն և որոնման ալգորիթմ |
| **JSON** | Տվյալների բազայի կառուցվածք |
| **HTML/CSS** | Վիզուալիզացիա և դիզայն |
